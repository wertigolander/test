## How to Deploy to Vercel

1. Create a GitHub account at github.com
2. Create a new repository
3. Upload these files to your repository
4. Go to vercel.com and sign up with your GitHub account
5. Click "New Project" and import your repository
6. Click "Deploy"

That's it! Your site will be live in under a minute.

## Adding More Projects

Just edit `index.html` and add new project cards in the `.projects` div!
